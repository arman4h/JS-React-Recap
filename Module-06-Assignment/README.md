# Module 06 Assignment - SwiftCart E-Commerce

## 📚 Overview

This was an assignment for **Mission One** - a comprehensive test of my intermediate JavaScript DOM manipulation and ES6 knowledge. The challenge was to build a fully functional e-commerce website using **raw HTML**, **Tailwind CSS**, and **vanilla JavaScript** without any frameworks.

## 🎯 Assignment Objective

The main goal was to create a cart-type e-commerce site where:
- All products are fetched dynamically from an API
- Products can be filtered by category
- Users can view product details in a modal
- Shopping cart functionality with add/remove items
- Responsive design for all devices

This assignment tested my understanding of:
- DOM manipulation
- ES6 features (arrow functions, template literals, async/await, etc.)
- API integration and data fetching
- Event handling
- Local storage management
- Responsive design principles

## 🛠️ Technology Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first styling (no custom CSS frameworks)
- **Vanilla JavaScript** - No frameworks like React, Vue, or Angular allowed
- **Fake Store API** - For product data
